## Command Details

### echo

Output text to stdout.

```bash
echo hello world
echo -n "no newline"
echo -e "escape\tsequences"
```

### printf

Formatted output with C-style format specifiers.

```bash
printf "%s %d\n" hello 42
printf "%.2f" 3.14159
```

### grep

Search text using patterns.

```bash
grep pattern file.txt
grep -i case-insensitive file.txt
grep -r recursive /path
```

### sed

Stream editor for text transformation.

```bash
sed 's/old/new/g' file.txt
sed -n '1,10p' file.txt
```

### awk

Pattern scanning and text processing.

```bash
awk '{print $1}' file.txt
awk -F: '{print $1}' /etc/passwd
```

### jq

JSON processor.

```bash
jq '.field' data.json
jq -r '.items[] | .name' data.json
```

### find

Search for files.

```bash
find /path -name '*.txt'
find . -type f -maxdepth 2
```

### tar

Archive operations.

```bash
tar -czf archive.tar.gz dir/
tar -xzf archive.tar.gz
```

### curl

HTTP client (requires network allowlist).

```bash
curl https://api.example.com/data
curl -X POST -d '{"key":"value"}' https://api.example.com
```
