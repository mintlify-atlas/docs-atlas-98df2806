## POSIX Shell Compliance

BashKit provides substantial compliance with IEEE Std 1003.1-2024 (POSIX.1-2024) Shell Command Language.

| POSIX Category | Status |

| Special Built-ins (15) | 13/15 implemented |
| Word Expansions | Substantial compliance |

**Security Exclusions:** `exec` and signal-based `trap` are intentionally not implemented for sandbox security.

## Intentionally Unimplemented

These features conflict with BashKit's stateless, virtual execution model:

| Feature | Rationale |
|---------|----------|
| `exec` builtin | Cannot replace shell process in sandbox |
| Background execution (`&`) | Stateless model - no persistent processes |
| Job control (`bg`, `fg`, `jobs`) | Requires process state |
| Symlink following | Prevents symlink loop attacks |
| Process spawning | External commands run as builtins |
| Raw network sockets | Only allowlisted HTTP via curl |
