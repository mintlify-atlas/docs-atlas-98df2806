Get started with BashKit by running your first virtual bash script in Rust.

## Prerequisites

- Rust 1.70 or later
- Basic familiarity with Rust async/await

## Installation

<Steps>
  <Step title="Add BashKit to your project">
    Add BashKit as a dependency to your Rust project:

    <CodeGroup>
    ```bash cargo add
    cargo add bashkit
    cargo add tokio --features full
    cargo add anyhow
    ```

    ```toml Cargo.toml
    [dependencies]
    bashkit = "0.1"
    tokio = { version = "1", features = ["full"] }
    anyhow = "1"
    ```
    </CodeGroup>
  </Step>

  <Step title="Create your first script">
    Create a simple program that executes bash commands:

    ```rust src/main.rs
    use bashkit::Bash;

    #[tokio::main]
    async fn main() -> anyhow::Result<()> {
        let mut bash = Bash::new();
        let result = bash.exec("echo hello world").await?;
        println!("{}", result.stdout); // "hello world\n"
        Ok(())
    }
    ```
  </Step>

  <Step title="Run it">
    ```bash
    cargo run
    ```

    Output:
    ```
    hello world
    ```
  </Step>

  <Step title="Try more commands">
    Bash scripts run in a virtual environment with a virtual filesystem:

    ```rust
    use bashkit::Bash;

    #[tokio::main]
    async fn main() -> anyhow::Result<()> {
        let mut bash = Bash::new();
        
        // Variables
        bash.exec("NAME=World").await?;
        let result = bash.exec("echo Hello, $NAME!").await?;
        println!("{}", result.stdout); // "Hello, World!\n"
        
        // Pipelines
        let result = bash.exec("echo -e 'apple\\nbanana\\ncherry' | grep a").await?;
        println!("{}", result.stdout); // "apple\nbanana\n"
        
        // Files (virtual filesystem)
        bash.exec("echo 'some data' > /tmp/file.txt").await?;
        let result = bash.exec("cat /tmp/file.txt").await?;
        println!("{}", result.stdout); // "some data\n"
        
        // Control flow
        bash.exec(r#"
            for i in 1 2 3; do
                echo "Number: $i"
            done
        "#).await?;
        
        Ok(())
    }
    ```
  </Step>
</Steps>

## Virtual Filesystem

All file operations happen in a virtual filesystem that exists only in memory:

```rust
let mut bash = Bash::new();

// Create a file in the virtual filesystem
bash.exec("mkdir -p /app/data").await?;
bash.exec("echo 'config' > /app/data/config.txt").await?;

// Read it back
let result = bash.exec("cat /app/data/config.txt").await?;
assert_eq!(result.stdout, "config\n");

// Files don't exist on your real filesystem!
```

<Note>
The virtual filesystem is isolated from your host system. Scripts cannot access your real files unless you explicitly mount them.
</Note>

## Common Operations

### Text Processing

```rust
// grep
let result = bash.exec("echo -e 'line1\\nline2\\nline3' | grep 2").await?;

// sed
let result = bash.exec("echo 'hello' | sed 's/hello/goodbye/'").await?;

// awk
let result = bash.exec("echo -e '1 2\\n3 4' | awk '{print $2}'").await?;

// jq
bash.exec("echo '{\"name\":\"Alice\",\"age\":30}' > data.json").await?;
let result = bash.exec("jq .name data.json").await?;
```

### Command Substitution

```rust
let result = bash.exec(r#"
    COUNT=$(echo -e "a\nb\nc" | wc -l)
    echo "Lines: $COUNT"
"#).await?;
println!("{}", result.stdout); // "Lines: 3\n"
```

### Error Handling

```rust
let result = bash.exec("cat /nonexistent").await?;

if result.exit_code != 0 {
    eprintln!("Command failed: {}", result.stderr);
} else {
    println!("Success: {}", result.stdout);
}
```

## Configuration

Use the builder to configure filesystem, environment, and limits:

```rust
use bashkit::{Bash, ExecutionLimits, InMemoryFs};
use std::sync::Arc;
use std::time::Duration;

let limits = ExecutionLimits::new()
    .max_commands(1000)
    .max_loop_iterations(10000)
    .timeout(Duration::from_secs(30));

let mut bash = Bash::builder()
    .fs(Arc::new(InMemoryFs::new()))
    .env("HOME", "/home/user")
    .env("PATH", "/usr/bin:/bin")
    .cwd("/home/user")
    .limits(limits)
    .build();

bash.exec("echo $HOME").await?; // "/home/user\n"
```

See the [configuration guide](/guides/configuration) for all options.
